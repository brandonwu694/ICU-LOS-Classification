# Sample Demo Data

These files are synthetic and are included only to make the public project notebook runnable in under 1 minute. They mimic the shape of the final model outputs without exposing restricted ICU records.

Files:
- `icu_los_classification_sample.csv`: synthetic first-24-hour ICU classification rows used by `notebooks/project.ipynb`.
- `demo_predictions.csv`: synthetic patient-level examples with observed and predicted ICU length of stay.
- `demo_model_metrics.csv`: representative final-model performance metrics.
- `demo_feature_importance.csv`: high-level feature groups used for interpretation.
- `demo_subgroup_error.csv`: subgroup error summary generated from the synthetic examples.

The `demo_*.csv` files are retained from the previous survival/log-LOS public demo.
